# Memory Game

## Description

- Simple CLI game to exercise Deno and TypeScript.

- Choose the number of users.

- Choose among six different cards and train your memory:

| Type    |
|---------|
| TIGER   |
| LEOPARD |
| WHALE   |
| FISH    |
| DOG     |
| CAT     |

- Enjoy the game.

## Running

```bash
deno task start
```

## Tests

```bash
deno task test:unit
```
